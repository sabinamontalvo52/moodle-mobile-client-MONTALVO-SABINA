# Moodle Mobile Client

Aplicación móvil desarrollada en Android (Kotlin) que consume la API REST de Moodle.

## Funcionalidades
- Autenticación mediante token
- Listado de cursos
- Visualización de actividades
- Envío de tareas
- Interacción con foros

## Tecnologías
- Android nativo (Kotlin)
- Retrofit
- Moodle REST API
